## Setup Node Enviroment
Install nodejs (v14 LTS)

https://nodejs.org/en/

Install yarn
https://classic.yarnpkg.com/en/docs/install#mac-stable

`npm install --global yarn`

Install http-server
https://www.npmjs.com/package/http-server

`npm install --global http-server`
